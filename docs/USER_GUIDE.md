# 청년동 도서대여 관리 운영자 안내서

배포 1차(Soft Launch)에서 내부 운영자가 매일 대여·반납을 처리할 때 보는 1페이지 안내서입니다.

## 접속 전 확인

- 접속 URL은 내부 운영자에게만 공유합니다.
- 운영자 접근 코드 화면이 나오면 담당자에게 공유받은 접근 코드를 입력합니다.
- 접근 코드는 접속 URL과 같이 내부 운영자에게만 공유합니다.
- 모바일 카메라 스캔은 HTTPS 배포 주소에서만 정상 동작합니다.
- 대여·반납 처리는 Google Sheets 원본 데이터에 바로 반영됩니다.
- 새 도서 추가나 시트 구조 변경은 웹앱이 아니라 Google Sheets 관리자 화면에서만 진행합니다.

## 기본 화면

- 상단 통계: 전체 도서, 대여 가능, 대여 중, 연체, 오늘 반납 예정 수를 확인합니다.
- 왼쪽 대여 등록: 회원/비회원 정보와 도서를 입력해 대여를 등록합니다.
- 오른쪽 도서 목록: 도서 검색, 상태 필터, 더보기, 반납 처리를 수행합니다.
- 최근 대여: 왼쪽 하단에서 최근 대여 중인 책을 빠르게 반납할 수 있습니다.

## 대여 등록

1. `회원` 또는 `비회원`을 선택합니다.
2. 이름과 전화번호를 입력합니다. 하이픈이나 공백은 입력해도 됩니다.
3. 도서 선택 방식을 고릅니다.
   - `바코드`: 카메라 스캔 또는 `CNDB`로 시작하는 코드를 직접 입력합니다.
   - `도서명`: 대여 가능한 도서명 목록에서 선택합니다.
4. 대여일과 반납예정일을 확인합니다.
5. `대여 등록하기`를 누르고 완료 메시지를 확인합니다.

## 반납 처리

1. 도서 목록에서 대여 중 또는 연체 상태인 책을 찾습니다.
2. 해당 행의 `반납` 버튼을 누릅니다.
3. 완료 메시지와 통계/목록 상태 변경을 확인합니다.

빠른 반납이 필요하면 왼쪽 하단 `최근 대여 · 빠른 반납` 영역에서 바로 `반납`을 누릅니다.

## 검색과 필터

- 검색어는 도서명, 저자, 대여자명, 도서 코드에 적용됩니다.
- 상태 필터는 전체, 대여 가능, 대여 중, 연체를 구분합니다.
- 목록이 길면 하단 `더보기`로 표시 개수를 늘립니다.

## 오류가 날 때

- `도서 바코드를 스캔해주세요.`: `CNDB`로 시작하지 않는 코드입니다. 바코드를 다시 스캔합니다.
- `반납예정일은 대여일보다 빠를 수 없습니다.`: 날짜를 다시 확인합니다.
- 목록 로드나 대여/반납 실패 메시지: 잠시 후 새로고침하고 다시 시도합니다.
- 같은 문제가 반복되면 운영 담당자에게 접속 URL, 발생 시각, 수행한 작업, 화면 캡처를 전달합니다.

## 운영 중 주의사항

- 전화번호는 운영 목적 외에 별도로 복사하거나 공유하지 않습니다.
- URL이 외부에 공유되었다고 의심되면 즉시 운영 담당자에게 알립니다.
- 반납 실수 취소 기능은 아직 없습니다. 실수 시 Google Sheets 원본에서 별도 확인이 필요합니다.
