# 청년동 도서관리 시스템 UI 와이어프레임 v1

## 목적
도서관리 시스템의 메인 화면 구조를 명확히 정의하고, 이후 UI 리뉴얼 시 기능 로직을 건드리지 않고 화면 구조만 개선하기 위한 기준 문서입니다.

---

# 기본 화면 구조

Hero
↓
Stats Cards
↓
대여 등록 | 도서 목록

---

# 1. Hero Section

청년동 도서관리
도서 대여부터 반납까지 한 화면에서 처리하세요.

회원 조회, 바코드 스캔, 대여·반납 현황 관리를 한 번에 진행할 수 있습니다.

# 2. Stats Section

- 전체 도서
- 대여 가능
- 대여 중
- 연체
- 오늘 반납 예정

Desktop: 5개 카드 가로 배치
Tablet: 3 + 2 배치
Mobile: 2열 또는 1열 배치

# 3. Borrow Panel

회원/비회원 선택
↓
도서 선택
↓
카메라 스캔
↓
이름/연락처 입력
↓
대여일/반납일 입력
↓
대여 처리

원칙:
- self-start 유지
- 단계별 흐름 명확화
- 가장 중요한 영역

# 6. Interaction Flow

회원 대여:
회원 선택 → 이름 입력 → 전화번호 입력 → 도서 선택 → 대여 처리

비회원 대여:
비회원 선택 → 이름 입력 → 전화번호 입력 → 도서 선택 → 대여 처리

반납:
도서 목록 → 반납 처리 → 시트 업데이트
